Pętla for

Konstrukcja: for (inicjalizacja; warunek; zmiana) { } Opis:

Pętla for jest używana, gdy z góry znamy liczbę powtórzeń. W każdej iteracji wykonuje się blok kodu, a po nim następuje aktualizacja zmiennej sterującej.

Pętla while (kontrolowane wejście)

Konstrukcja: while (warunek) { } Opis:

Pętla while sprawdza warunek przed każdą iteracją. Jeśli warunek jest prawdziwy, wykonywany jest kod wewnątrz pętli. Może nie wykonać się ani razu, jeśli warunek jest od początku fałszywy.

Schemat blokowy:

Pętla do...while (kontrolowane wyjście)

Konstrukcja: do { } while (warunek); Opis:

Pętla do...while wykona się co najmniej raz, ponieważ warunek jest sprawdzany po wykonaniu bloku kodu. Jest to tzw. pętla z kontrolą u dołu.
